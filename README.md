Custom fork of Camera Tools Continued adding an API surface to be used by Cinematic Recorder.

This project attempts to change as little about the original code as possible, aiming to insert the API in the least disruptive way.  The API expands the original rather than altering the core or base. 
This version of Camera Tools is required for Cinematic Recorder to be able to work with Camera Tools cameras.
